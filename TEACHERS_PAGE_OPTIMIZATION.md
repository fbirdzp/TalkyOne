2. **科目**：选择要学习的科目

- 点击页码按钮跳转

## 常见问题

### 1. 页面显示 404
**解决**：
```bash
# 重启开发服务器
pkill -f "next dev"
npm run dev
```

### 2. Slider 组件不工作
**原因**：未安装 `@radix-ui/react-slider`
**解决**：
```bash
npm install @radix-ui/react-slider
```

### 3. 筛选没有效果
**原因**：API 未正确接收参数
**解决**：
- 检查浏览器开发者工具中的网络请求
- 确认 API 返回的 `total` 数量是否正确
- 查看 `.next/logs` 中的错误日志

### 4. 分页不工作
**原因**：`totalPages` 计算错误
**解决**：
- 检查 API 返回的 `totalPages` 是否正确
- 确认 `limit` 参数是否设置为 20
