## 常见问题排查

### 1. 数据库问题

**问题**: Prisma 无法连接到数据库

**解决方案**:

```bash
# 确保 PostgreSQL 正在运行
brew services start postgresql

# 创建数据库
createdb talkyone

# 运行 Prisma migration
npx prisma migrate dev

# 生成 Prisma Client
npx prisma generate
```

### 2. 端口占用问题

**问题**: 端口 3000 已被占用

**解决方案**:

```bash
# 查找占用端口的进程
lsof -ti:3000

# 杀死进程
kill -9 <PID>

# 或者修改 package.json 中的 dev 脚本
"dev": "next dev -p 3001"
```

### 3. 依赖安装问题

**问题**: npm install 失败

**解决方案**:

```bash
# 清除 npm 缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装依赖
npm install
```

### 4. ESLint 错误

**问题**: 代码有 ESLint 错误，无法提交

**解决方案**:

```bash
# 自动修复 ESLint 错误
npm run lint:fix

# 或者手动修复错误
# 查看 ESLint 错误信息，手动修复代码
```
